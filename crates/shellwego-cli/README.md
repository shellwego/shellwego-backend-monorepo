# ShellWeGo CLI
**The Hacker's Interface.** Zero-BS deployment from the terminal.

- **Speed:** Deploy apps, scale replicas, and provision DBs in seconds.
- **TUI:** Built-in `top` command using `ratatui` for real-time cluster monitoring.
- **Auth:** Secure token management via system keyring.
- **SSH:** Transparent debugging access to microVMs.
